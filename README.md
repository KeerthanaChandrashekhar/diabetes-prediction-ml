# 🧠 Diabetes Prediction using Machine Learning

## 📌 Project Overview

This project uses Machine Learning algorithms to predict whether a person has diabetes based on health-related features.

The goal is to evaluate whether **K-Nearest Neighbors (KNN)** is suitable for this dataset and compare its performance with other models.

---

## 📊 Dataset

Dataset: **Diabetes Prediction Dataset**

Features include:

* Age
* Gender
* BMI
* Hypertension
* Heart Disease
* Smoking History
* HbA1c Level
* Blood Glucose Level

Target variable:

* **diabetes (0 = No, 1 = Yes)**

Dataset Source: Kaggle
